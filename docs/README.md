# Kit
A multi-backend platform layer.

# Roadmap and wishlist
- [ ] Single-header C file.
- [ ] Write once, deploy many.
- [ ] Try different backends easily.
- [ ] For web, desktops, consoles and mobiles.
- [ ] Subsystems:
  - [ ] Editor widgets.
  - [ ] Windowing and surface contexts.
  - [ ] Keyboard, gamepads, mouse and cursors.
